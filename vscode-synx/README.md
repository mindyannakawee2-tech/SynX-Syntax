# SynX VS Code Support

This extension scaffold provides:

- Syntax highlighting for `.synx`
- Auto-closing and bracket pairing
- Snippets (`set`, `work`, `if`)
- Basic diagnostics warning when Lua keywords (`local`, `function`) are used

To package:

1. `npm i -g vsce`
2. `cd vscode-synx`
3. `vsce package`
